The HejBit alternative

There are a few commonly accepted misconceptions about data sovereignty.

1

Misconception: centralised data is secure.

Data breaches, cloud storage failures, unauthorized access, data mining, and policy changes by third-party providers gets increasingly worse.

2

Misconception: decentralised data solutions are complicated.

The advantages of using blockchain technology for ultimate data privacy come with the disadvantages of immense technical complexity for the end user.

3

Misconception: centralised dropboxes and cloud drives offer a better user experience.

Centralisation and decentralisation of data is back-end technologies. For management of the data the frontend can be just as convenient.
