Feature List

Seamless integration with Nextcloud.

Swarm file Management: Upload, Download, Folder Management.

Swarm file Sharing Capabilities: Private link, Public Link.

Swarm file permissions: Note to Recipient, Read / Update / Delete, Can Edit, Password, Expiration Date, Hide Download

Swarm file Viewing & Streaming: Text files, Images, Documents (Word, spreadsheets), Audio files, Video files, PDF files.

Swarm Hash Features: Check Swarm Hash, Selectively hide Swarm Hash for defined users/groups, Swarm Hash backup.

Identity management & Authentication: LDAP, CAS, Auth0, SAML, Shibboleth, Kerberos, amongst serveral other options.

Access Control: Detailed settings for users, groups and guest access using Swarm Storage.

Data Migration: From Google Drive, One drive, iCloud, DropBox to Nextcloud/Swarm Storage.
